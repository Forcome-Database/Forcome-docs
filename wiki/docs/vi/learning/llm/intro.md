# LLM là gì

Mô hình ngôn ngữ lớn là mô hình học sâu cho NLP.

## Khái niệm cốt lõi

- **Transformer** - Kiến trúc mô hình
- **Pre-training** - Học từ dữ liệu lớn
- **Fine-tuning** - Tối ưu cho tác vụ cụ thể
- **Inference** - Tạo văn bản

## Các mô hình phổ biến

- GPT (OpenAI)
- Claude (Anthropic)
- Llama (Meta)
