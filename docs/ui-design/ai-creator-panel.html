模式三：对话模式（仅面板内对话）

✨

AI 创作

💬 选中文本
G1 性能——峰值 QPS 下 P95 延迟 ≤ X ms；G2 扩展性——可水平扩展至 N 倍流量；
你
用通俗的语言解释一下这些目标的含义
AI 2.1s
这三个目标的含义是：

G1 性能 — 系统必须响应够快。在峰值负载下，95% 的请求应在 X 毫秒内完成。

G2 扩展性 — 系统可以通过增加服务器来承载更多流量，无需重新设计架构。

G3 合规性 — 系统满足处理敏感数据所需的国际安全标准。
你
X ms 填什么值比较合理？
AI 1.5s
对于大多数 Web 服务，合理的 P95 延迟目标是 200-500ms。实时 API 应控制在 <100ms。批处理场景下 1-2s 可以接受。

设计说明

  • 对话模式：AI 仅在面板内回答，不修改编辑器
  • 每条 AI 消息都有复制 + 插入按钮
  • 用户可按需将任意 AI 回答插入编辑器
  • 支持多轮上下文对话
